# 🚀 Algorithm Visualizer

A web-based interactive tool built to help developers and students visualize how complex algorithms work under the hood. 

This project brings abstract math to life through clean animations and real-time state tracking. Watch arrays get sorted step-by-step or see how pathfinding algorithms navigate a grid!

## ✨ Features
* **Sorting Algorithms:** Bubble Sort, Quick Sort, Merge Sort, etc.
* **Pathfinding Algorithms:** Dijkstra's, A* Search (Coming Soon!)
* **Interactive Controls:** Adjust sorting speed, generate random data sets, and pause/play execution.
* **Color-Coded Feedback:** Visually track elements as they are compared, swapped, and finalized.

## 🛠️ Tech Stack
* **Frontend:** React.js / Vite
* **Styling:** CSS / TailwindCSS
* **Deployment:** Vercel
